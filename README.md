# Jogo da Velha

## O que é?

O jogo da velha (também conhecido como Tic-Tac-Toe) é um jogo de estratégia simples para dois jogadores, jogado em um tabuleiro de 3x3.

## Como funciona?

1. **Tabuleiro**: Uma grade de 3x3 (9 posições no total)
2. **Jogadores**: Um jogador usa "X" e o outro usa "O"
3. **Turnos**: Os jogadores alternam turnos, marcando uma posição vazia por vez
4. **Objetivo**: Ser o primeiro a formar uma linha, coluna ou diagonal com 3 marcas iguais

## Condições de vitória

- 3 marcas na mesma linha (horizontal)
- 3 marcas na mesma coluna (vertical)
- 3 marcas na mesma diagonal

## Resultados possíveis

- Vitória de um dos jogadores
- Empate (quando o tabuleiro fica cheio sem vencedor)

## Características

- Simples de aprender
- Rápido de jogar
- Com jogo perfeito, sempre termina em empate
- Popular como jogo de papel e lápis

É um jogo clássico usado para ensinar programação e lógica, incluindo em Flutter para criar interfaces e gerenciar estado.
